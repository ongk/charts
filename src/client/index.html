Donut Chart Directive

This directive generates a simple donut chart using D3. Any changes to chart-data will fire an update to the chart.

  • Markup 
    <div ng-app="hz.app" ng-controller="MyCtrl">
      <donut-chart chart-data="data" chart-text="25%"></donut-chart>
    </div>
    Attributes:
    • chart-data (required) - attribute name of data defined in $scope
    • chart-diameter - diameter of donut chart
    • chart-thickness - thickness of donut
    • Any attributes for <text> element such as font-size, font-family, fill, etc...
  • Controller 
      angular.module('app').controller('MyCtrl', [ '$scope', function($scope) {
    $scope.instances = {
      data: [
        { label: 'Current', slice: 1 },
        { label: 'Applied', slice: 1 },
        { label: 'Available', slice: 6 }
      ],
      colors: [ '#1f83c6', '#81c1e7', '#d1d3d4' ],
      text: "25%"
    };

    // Change the data (optional)
    $scope.updateChart = function() {
      $scope.instances = {
        data: [
          { label: 'Current', slice: 1 },
          { label: 'Applied', slice: 7 }
        ],
        colors: [ '#1f83c6', '#ff0000' ],
        text: "100%"
      };
    };
  }]);
DescriptionCodeRendered
Default 
<donut-chart chart-data="instances"></donut-chart>
Custom diameter, thickness, font size 
<donut-chart chart-data="vcpus" chart-diameter="150" chart-thickness="20" font-size="3em"></donut-chart>
Using Font Awesome icon 
<donut-chart chart-data="ram" chart-diameter="150" chart-thickness="20" font-family="FontAwesome" font-size="3.5em"></donut-chart>
