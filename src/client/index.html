Donut Chart Directive

Required attributes:
  • chart-data - attribute name of data defined in $scope
  • chart-text - text to display in donut hole
Optional attributes:
  • chart-colors - attribute name of colors defined in $scope
  • chart-diameter - diameter of donut chart
  • chart-thickness - thickness of donut
  • Core "text" attributes such as font-size, font-family, fill, etc...
  • HTML 
    <donut-chart chart-data="data" chart-text="25%"></donut-chart>
  • Controller 
      angular.module('app').controller('MyCtrl', [ '$scope', function($scope) {
    $scope.data = [
      { label: 'Current', portion: 1 },
      { label: 'Applied', portion: 1 },
      { label: 'Available', portion: 6 }
    ];
    $scope.colors = [ '#0000ff', '#00ff00', '#999' ];
  }]);
DescriptionCodeRendered
Default 
<donut-chart chart-data="data" chart-text="25%"></donut-chart>
Custom colors 
<donut-chart chart-data="data" chart-text="25%" chart-colors="colors"></donut-chart>
Custom diameter, thickness, font size 
<donut-chart chart-data="data" chart-diameter="150" chart-thickness="20" chart-text="25%" font-size="2em"></donut-chart>
Using Font Awesome icon 
<donut-chart chart-data="data" chart-diameter="150" chart-thickness="20" chart-text="f207" chart-is-unicode="true" font-family="FontAwesome" font-size="2em"></donut-chart>
